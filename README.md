# AtWork — тестовое задание

Двухстраничное приложение на React для управления пользователями.

## Стек

- React 18 + TypeScript
- React Router v7
- TanStack Query v5
- Zustand
- React Hook Form + Zod
- SCSS Modules
- Vite

## Запуск

```bash
npm install
npm run dev
```

Откроется на [http://localhost:5173](http://localhost:5173).

## Сборка

```bash
npm run build
npm run preview
```

## Что умеет

**Главная страница** — список первых 6 пользователей с JSONPlaceholder. На каждой карточке никнейм, компания, город и аватарка. Через меню ⋮ можно отредактировать, архивировать или скрыть пользователя. Архивные карточки отображаются отдельной секцией, оттуда можно вернуть пользователя в активные.

**Страница редактирования** — форма с полями: имя, никнейм, почта, город, телефон, компания. Есть валидация (длина полей, формат email, только цифры в телефоне). После сохранения появляется попап, который закрывается сам через 4 секунды или по клику.
